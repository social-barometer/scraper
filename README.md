# scraper
Scrapes internet for data to be displayed in the app

## Installation
`npm install`

## Usage
`npm start`

## Testing
`npm test`

## Code Coverage
`npm run coverage`

## Job dashboard
Manage jobs with a nice GUI
`npm run agendash`

## Documentation
Generate docs
`npm run docs`

### Example
Example client implementation in `client.js`
